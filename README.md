# Link Directory

A simple, elegant project directory manager to organize and access your links.

## Features

- Add, edit, and delete projects with names and URLs
- Star favorites for quick access
- Drag and drop to reorder projects
- Click rows to open links in new tabs
- Auto-fetch favicons for visual recognition
- Clean, modern UI with gradient styling

## Setup

```bash
npm install
npm start
```

Open `http://localhost:9000` in your browser.

## Tech Stack

- Node.js + Express backend
- Vanilla JavaScript frontend
- File-based JSON storage
